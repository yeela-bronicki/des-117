Flexbox Challege 9

Set the section to display flex, give the flex items a 20px gap and make sure all three paragraphs are the same width. Then use the cross axis alignment to change the vertical alignment of the flex items (the paragraphs in this case). Align them to the bottom of the section.

Note that these cross axis alignments work because there's a minimum height to the section, which is taller than the contents in the section.
